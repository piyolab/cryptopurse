# cryptopurse
cryptopurse.app

## Development
### Tools installation
```
npm install gulp-cli -g

# Optional
npm install pug-cli -g
npm install browser-sync -g
```

### Gulp
```
npm install
gulp
```

### Pug
```
pug pug -o public -P -w
```

### Browser sync
```
browser-sync start --server 'public'
```

### Use testnet
- add the following parameter

```
?network=3
```
